# GALauncher (GAL)

**다클라이언트 자동 로그인 및 실행 런처**

이 프로젝트는 거상 게임의 다중 클라이언트 실행 편의를 위해 개발된 **비공식 런처**입니다.
C# WinForms (.NET 8.0) 기반으로 제작되었으며, WebView2를 활용하여 안정적인 웹 로그인과 자동 실행 시퀀스를 제공합니다.

## 🚀 주요 기능

* **다클라이언트 원클릭 실행:** 최대 3개의 클라이언트를 설정된 순서대로 자동 실행.
* **개별/전체 제어:** 각 클라이언트별 개별 실행 및 전체 자동 실행 지원.
* **웹 로그인 지원:** WebView2 기반의 안전한 로그인 방식 적용.

## 🛠️ 설치 및 실행 방법

### 1. 다운로드
오른쪽 **[Releases]** 탭에서 최신 버전(`GALauncher.exe`)을 다운로드합니다.

### 2. SmartScreen 경고 해결
개인 개발자가 만든 프로그램이므로 실행 시 **"Windows의 PC 보호"** 파란색 OR 빨간색 경고창이 뜰 수 있습니다.
* **해결:** [추가 정보] → [실행] 버튼을 클릭하면 정상적으로 실행됩니다.

### 3. Antivirus(백신) 경고 해결
일부 백신에서 오진할 경우 실행이 차단될 수 있습니다.
* **해결:** 백신 프로그램의 검사 예외 항목(검역소 등)에서 복구 및 예외 처리를 해주세요.

### 4. 초기 설정
1.  **클라설정:** 거상이 설치된 폴더 3곳(A, B, C)의 경로를 지정합니다.
2.  **계정설정:** 사용할 계정 아이디와 비밀번호를 입력하고 저장합니다.
3.  **시작:** '세트 추가'를 눌러 원하는 조합(아이디+클라이언트)을 구성합니다.
4.  **[전체실행]** 버튼을 누르면 순차적으로 게임이 켜집니다.
5.  **[외부]** 체크 후 [홈피] 버튼을 누르면 별도 브라우저에서 로그인이 됩니다.
